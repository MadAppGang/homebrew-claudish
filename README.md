# Homebrew Tap for claudish

Run Claude Code with any OpenRouter model.

## Installation

```bash
brew tap MadAppGang/claudish
brew install claudish
```

## Usage

```bash
claudish                  # Interactive mode
claudish --model <name>   # Use specific model
claudish --mcp            # MCP server mode
```
